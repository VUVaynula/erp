Документ «Форма сбора данных»

 

Документ «Форма сбора данных» предназначен для фиксации в системе произвольных данных и движений по регистрам. С помощью данного документа в систему могут быть загружены данные из файлов Excel.

 

В документе указывается «Вид формы сбора данных» (выбирается из одноименного справочника), при этом на первую закладку документа добавляются реквизиты шапки, а на следующие закладки – реквизиты табличных частей, настроенные в выбранном виде формы сбора данных. После этого документ готов для заполнения данных или загрузки их из файла Excel.

 

При проведении документа будут сформированы движения по регистрам в соответствие с настроенными в виде формы сбора данных шаблонами движений.

 

Загрузка данных из Excel

 

Для загрузки в систему произвольных данных из файлов Excel используется справочник «Виды форм сбора данных» и документ «Форма сбора данных».

Созданный в справочнике «Виды форм сбора данных» шаблон документа по кнопке «Выгрузить шаблон» выгружается в файл Excel.

 

Возможна групповая выгрузка шаблонов из справочника «Виды форм сбора данных». Для этого необходимо в списке справочника выделить несколько элементов с помощью кнопки Shift и нажать кнопку «Выгрузить шаблон». В этом случае для каждого вида форм сбора данных в файле Excel будет создан отдельный лист.

На отдельный лист выгружаются списки значений справочников, планов счетов и других объектов ссылочного типа - в соответствии с настройкой вида форм сбора данных. При заполнении формы выбирать такие значения нужно будет из выпадающего списка.

 

После заполнения данных файл необходимо сохранить.

 

Загрузка данных из файла Excel  в систему происходит в форме списка документов «Форма сбора данных» или непосредственно в форме документа «Форма сбора данных» по кнопке «Загрузить данные».

Если загрузка происходит в форме документа «Форма сбора данных», то по кнопке «Загрузить данные» откроется стандартное окно для выбора  нужного файла с данными. При выборе файла в документ автоматически будут загружены из файла данные, соответствующие указанному виду формы сбора данных.

Из формы списка можно загрузить данные как одного, так и нескольких файлов Excel. По кнопке «Загрузить данные» открывается форма «Загрузка форм сбора данных», в которой нужно указать путь к каталогу, в котором хранятся файлы для загрузки, или к конкретному файлу.

 

Все данные, хранящиеся в выбранном каталоге или файле, будут проанализированы системой, и в табличную часть формы будут выведены файлы и листы, соответствующие определенным видам форм сбора данных. Далее флажками следует выделить файлы и/или листы, которые должны быть загружены, и нажать кнопку «Загрузить».

В системе при этом будут созданы документы «Форма сбора данных». В форме «Загрузка форм сбора данных» на закладке «Загруженные объекты» можно просмотреть созданные документы и, если данные загрузились некорректно, можно удалить документы по кнопке «Удалить объекты».

Если данные загружены корректно, форма «Загрузка форм сбора данных» может быть закрыта. Документы «Форма сбора данных» необходимо провести для фиксации данных в системе.

 

Примечание. Рекомендуемый режим работы с формами сбора данных - не более 1000 строк / 10000 заполненных значений.
