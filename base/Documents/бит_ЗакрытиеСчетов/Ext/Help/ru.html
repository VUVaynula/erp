Документ «Закрытие счетов»

 

 1.      Назначение документа

 

 Документ «Закрытие счетов» предназначен для формирования движений закрытия счетов по заданным механизмам.

 

 2.       Процедура заполнения

 

 По кнопке «Заполнить» на закладке «Закрытие» табличная часть будет автоматически заполнена движениями по закрытию счетов. В каждой строке табличной части можно изменить значение любой их аналитик и суммы.

 

Очищение табличной части происходит по кнопке  «Очистить», расположенной на панели инструментов.

 

 3.      Описание реквизитов

 

В шапке документа необходимо указать Организацию и Вид закрытия. Регистр бухгалтерии будет заполнен в документе  по виду закрытия, данное соответствие прописано в справочнике «Виды закрытия счетов». Автоматически будет заполнена настройка закрытия на закладке «Дополнительно». Значение будет получено на основе установленного вида закрытия и регистра сведений «Назначение алгоритмов закрытия».

 

В табличной части закладки «Заполнение» отражены поля:


  • Счет – счет, подлежащий закрытию;
  • Аналитика счета, подлежащего закрытию;
  • Счет закрытия -  счет, на который будут распределены  остатки (обороты);
  • Сторона закрытия – поле заполнено, если закрываемый счет активно-пассивный, в справочнике «Настройки закрытия» регистрируется сторона закрытия;
  • Аналитика счета закрытия;
  • Количество – поле заполняется, если счет количественный;
  • Валюта, Валютная сумма – заполняются, если счет валютный;
  • Сумма – сумма подлежащая распределению на счет закрытия;
  • Содержание -  текст поля определяется в справочнике «Настройки закрытия»;
  • № журнала - текст поля определяется в справочнике «Настройки закрытия».

 

По необходимости фиксируется комментарий к документу в текстовом поле «Комментарий». В поле Ответственный проставляется пользователь, под чьим сеансом создан документ.
