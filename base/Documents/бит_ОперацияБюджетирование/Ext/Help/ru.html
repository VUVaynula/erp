Документ «Операция (бюджетирование)»

 

Документ «Операция (бюджетирование)» предназначен для ввода начальных остатков по счетам бюджетирования.

 

Начальные остатки вводятся отдельно по каждому сценарию бюджетирования в корреспонденции со вспомогательным счетом 000.

 

Правильность внесения остатков по счетам бюджетирования можно проверить с помощью стандартного отчета «Оборотно-сальдовая ведомость (бюджетирование)», сформировав его по каждому сценарию, по которому вносились остатки. На вспомогательном счете 000 сальдо на дату ввода остатков должно получиться равным нулю.

 

Кроме ввода начальных остатков, с помощью документа «Операция (бюджетирование)» можно формировать ручные проводки и корректировать уже введенные плановые и фактические данные по счетам плана счетов бюджетирования.
