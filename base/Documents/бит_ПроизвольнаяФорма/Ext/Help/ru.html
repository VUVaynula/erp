Документ «Произвольная форма»

 

Документ «Произвольная форма» предназначен для фиксации в системе произвольных данных и движений по регистрам. 

 

В документе указывается «Вид произвольной формы» (выбирается из одноименного справочника), при этом на первую закладку документа добавляются реквизиты шапки, а на следующие закладки – реквизиты табличных частей, настроенные в выбранном виде формы сбора данных.

При проведении документа будут сформированы движения по регистрам в соответствие с настроенными в виде формы сбора данных шаблонами движений.


Примечание. Рекомендуемый режим работы с произвольной формой - не более 1000 строк / 10000 заполненных значений.
