Документ «Изменение параметров учета ОС (МСФО)»

 

Документ «Изменение параметров учета ОС (МСФО)» предназначен для изменения параметров основных средств.

 

В табличной части документа указывается основное средство. Для указанных объектов основных средств автоматически заполняются действующие эксплуатационные параметры.  Новые значения бухгалтерских оценок пользователь заполняет вручную.


После проведения документа, новые эксплуатационные параметры учета ОС попадают в регистр сведений «Параметры ОС (МСФО)».  Бухгалтерских проводок по международному учету документ не формирует.
