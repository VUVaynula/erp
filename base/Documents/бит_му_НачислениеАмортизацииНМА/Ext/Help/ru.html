Документ «Начисление амортизации НМА (МСФО)»

 

Документ «Начисление амортизации НМА (МСФО)» предназначен для начисления амортизации нематериальных активов в международном учете. 


Пользователь может заполнить табличную часть автоматически или вручную. Для автоматического заполнения необходимо воспользоваться кнопкой «Подбор». В табличную часть будут подобраны все нематериальные активы, по которым необходимо начислить амортизацию. Кроме наименования и сумы будут отражены для удобства пользователя параметры, влияющие на расчет суммы амортизации.  


В «БИТ.ФИНАНС: Управленческий учет» реализованы три  метода начисления амортизации: линейный метод, метод уменьшаемого остатка и производственный метод.
