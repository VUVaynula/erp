Документ «Установка активных комплектов движений»

 

1.      Назначение документа

 

Документ «Установка активных комплектов движений» предназначен для установки активности комплектов движений, как для одной организации, так и для всех организаций сразу. Существует возможность «периодической установки», т.е. на протяжении одного периода будет активным один комплект, на протяжении другого – другой комплект. Возможность одновременного использования для одной и той же организации в один и тот же период нескольких комплектов – исключена.

 

 

2.      Процедура заполнения

 

Документ заполняется вручную пользователем.

 

3.      Описание реквизитов

 

Назначение реквизитов и элементов формы:

·      Организация - организация, для которой устанавливается активность комплектов движений. При незаполненной организации, активность устанавливается для всех организаций;

·        Подбор - кнопка, вызывающая открытие формы подбора комплектов движений;

·       Вид документа - реквизит табличной части, в котором указывается, для какого вида документа устанавливается активным комплект движений;

·        Комплект – активный комплект движений.

 

4.      Движения документа при проведении

 

После проведения документ делает записи в регистр сведений «Активные комплекты движений».
