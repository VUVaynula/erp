Документ «Получение фактических данных»

 

Документ «Получение фактических данных» (меню «Бюджетирование» - «Получение фактических данных») предназначен для формирования движений по факту в регистре «Обороты по бюджету» по данным РСБУ.  Пользователь может заполнить фактические данные вручную или автоматически по настроенным источникам данных (кнопка «Заполнить»).

 

В документе указывается Сценарий, на который будут записаны фактические данные (это может быть предопределенный сценарий «ФАКТ» или любой другой сценарий), Период, за который заполняются фактические данные, Валюта и Курс (по умолчанию берется на дату документа).

 

ЦФО, Статья оборотов и Проект в шапке документа заполняются только при использовании механизма разграничения прав доступа на уровне записей (RLS).

 

На закладке «Источники данных» указываются источники данных – список доступных источников данных из одноименного справочника, у которых установлен тип «Получение факта».

 

После заполнения пользователь может отредактировать фактические данные.

 

При проведении документа фактические данные по бюджету будут сохранены в подсистеме бюджетирования.

 

На закладке «Зависимые обороты» отражаются данные по зависимым статьям оборотов, если зависимость была настроена ранее в документе «Установка зависимых оборотов». На данной закладке можно указать формировать или нет зависимые обороты при проведении документа.

 

На закладке «Прочее» можно указать, что при проведении документа следует формировать фактические данные только по зависимым оборотам (флаг «Формировать движения только по зависимым оборотам»). Это может быть полезно в ситуации, когда фактические данные по основным статьям оборотов (статьям – источникам) были получены ранее, например, при помощи механизма трансляции.

 

В случае, если в статьях оборотов настроены шаблоны проводок, но при проведении документа не требуется формирование фактических данных по плану счетов бюджетирования, можно установить флаг «Не формировать проводки бюджетирования».
