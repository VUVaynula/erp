Документ «Изменение параметров учета НМА (МСФО)»

 

Документ «Изменение параметров учета НМА(МСФО)» предназначен для изменения параметров нематериальных активов.

 

В табличной части указывается нематериальный актив. Для каждого объекта автоматически заполняются действующие эксплуатационные параметры, пользователь вносит новые измененные значения бухгалтерских оценок. 


После проведения документа, новые эксплуатационные параметры учета НМА попадают в регистр сведений «Параметры НМА (МСФО)». Бухгалтерских проводок по международному учету документ не формирует.
