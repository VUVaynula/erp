Документ «Переоценка НМА (МСФО)»


По кнопке «Подбор» в дополнительной табличной части будут отражены все нематериальные активы, учитывающиеся по справедливой (переоцененной) стоимости.


Для каждого нематериального актива указывается новая справедливая стоимость. Текущая балансовая стоимость будет автоматически рассчитана системой. Счета доходов и расходов в табличную часть отражаются из регистра сведений «Параметры классов ВНА (МСФО)» и могут быть изменены в табличной части документа. Счет резерва переоценки для формирования движений берется из регистра сведений «Параметры классов ВНА (МСФО)».
