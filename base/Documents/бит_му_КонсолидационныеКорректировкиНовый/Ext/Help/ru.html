Документ «Консолидационные корректировки»


Документ «Консолидационные корректировки» предназначен для отражения консолидационных корректировок, необходимых для получения консолидированной отчетности: расчет показателя гудвилла для консолидированной отчетности, расчет показателя нераспределенной прибыли для консолидированной отчетности, расчет показателей капитала для консолидированной отчетности.


В шапке документа:


·        Номер (генерируется системно);

·        Дата (дата документа);

·        Период (период за который формируется консолидированная отчетность);

·        Периметр консолидации (список организаций, по которым формируется консолидированная отчетность, и создаются элиминационные проводки);

·        Организация (организация из периметра с типом «консолидирующая», на которую записываются все корректирующие движения);

·        Регистр бухгалтерии (регистр бухгалтерии, по которому формируется консолидированная отчетность);

·        Счет сторно(вспомогательный счет для сторнирующих движений).


На закладке «Гудвилл» формируются данные о показателе гудвилл для консолидированной отчетности по группе компаний, указанных в шапке документа. В табличной части фиксируется организация, сумма гудвилла на момент приобретения, на  дату начала периода и на конец периода. На основе этих данных формируются движения по регистру бухгалтерии, указанному в шапке документа.


Дт Счет учета  гудвилла Кт Счет вспомогательный (сумму начала периода по каждой организации);

Дт Счет списания гудвилла  Кт Счет учета гудвилла  (на разницу суммы, на дату начала и дату окончания).

 

На закладке «Нераспределенная прибыль»  рассчитывается показатель консолидированной нераспределенной прибыли. В табличном поле указываются списком счета учета нераспределенной прибыли (НРП), которые будут участвовать в анализе. В табличном поле определяем элементы консолидированной нераспределенной прибыли:


·        НРП материнской компании (остаток по счету учета НРП компании, у которой стоит флаг «Осуществляет контроль периметра» в элементе справочника «Периметры консолидации»);

·        НРП дочерней компании (оборот по счетам НРП за период, когда дочерняя компания входила в периметр, умноженный на долю владения);

·        НРП ассоциированной компании (оборот по счетам НРП за период, когда ассоциированная компания входила в периметр, умноженный на долю владения);

·        НРП совместно контролируемой компании (оборот по счетам НРП за период, когда совместно контролируемая компания входила в периметр, умноженный на долю владения);

·        Обесценение гудвилла(сумма обесценения гудвилла с момента приобретения до даты отчетности);

·        Удаление нереализованной прибыли (используется в случае, если не используется автоматический вариант удаления нереализованной прибыли по МПЗ);

·        Прочие корректировки (внесение ручных корректировок, необходимых для корректировки консолидированной нераспределенной прибыли);

·        Итого (сумма всех составляющих показателя консолидированной нераспределенной прибыли).


Движения, формируемые документом по отражению консолидированной нераспределенной прибыли:

Дт Вспомогательный счет КтСчет НРП (Сумма итого по табличной части).


На закладке «Капитал» определяются показатели капитала для консолидированной отчетности. Указываются счета учета акционерного капитала и счета учета эмиссионного дохода. В табличное поле выводятся данные материнской компании с возможностью ручной корректировки. Колонки табличной части:


·        Организация (организация, осуществляющая контроль периметра);

·        Составляющая капитала (перечисление: акционерный капитал, эмиссионный доход);

·        Счет;

·        Сумма.


Движения, формируемые документом по отражению капитала в консолидированной отчетности:


Дт Вспомогательный счет КтСчет капитала (Сумма по строке табличной части).


На закладке «Дополнительно»  можно добавить комментарий.
