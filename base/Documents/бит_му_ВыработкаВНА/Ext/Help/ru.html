Документ «Выработка ВНА (МСФО)»


1. Назначение документа

 

Документ «Выработка ВНА (МСФО)» предназначен для отражения количества произведенной продукции за период с участием определенного внеоборотного актива. Документ используется  для внеоборотных активов с производственным методом начисления амортизации.

 

Документ имеет два вида операции:


Основные средства - для отражения выработки по основному средству


Нематериальные активы - для отражения выработки по нематериальному активы

 

2. Процедура заполнения

 

Заполнение табличной части объектами ВНА возможно вручную или автоматически. По кнопке «Подбор» автоматически можно заполнить табличную часть активами, у которых установлен производственный метод амортизации. 

 

Вручную пользователем заполняется количество выработанной продукции для каждого объекта табличной части. Если за месяц внесено несколько документов «Выработка ВНА (МСФО)», то амортизация будет рассчитана на основе общего показателя произведенных единиц за период.

 

3.      Движения документа при проведении

 

При проведении документ формирует движения (приход) по регистру накопления «Выработка ВНА (МСФО)».
