Документ «Установка контрольных значений»

 

Документ «Установка контрольных значений» предназначен для установки значений, в соответствии с которыми будут контролироваться суммы расходов по  заявкам на расходование денежных средств, (или) суммы оборотов по отдельным бюджетам, а также

 

Документ имеет три вида операции - «Контроль заявок», «Контроль бюджета» и «Контроль графиков договоров» для установки соответствующих контрольных значений.

 

Реквизиты Вид отклонения (сум.) и Вид отклонения (кол.) предназначены для установки допустимых отклонений от контрольных значений бюджета в абсолютной или относительной величине. Значения отклонений указываются в полях Значение.

 

По умолчанию сумма заявки контролируется по бюджету в разрезе статьи оборотов. Если кроме этого необходимо контролировать сумму заявки с учетом других аналитик бюджетирования (ЦФО, Проект и т.п.), то необходимо напротив нужных аналитик установить флаг в колонке «Использовать» в поле «Настройка заполнения».

 

На закладке «Настройка отбора» можно установить отбор по какой-либо аналитике (Статья оборотов, ЦФО и др.) или реквизиту аналитики (Вид статьи, Вид ЦФО и др.), для которых необходимо задать свое значение отклонения.

 

При проведении документа все бюджетные данные будут зафиксированы в специальном регистре накопления «Контрольные значения бюджетов». Это и будут контрольные значения, с которыми будут сравниваться суммы заявок на расходование денежных средств и заявок на затраты, данные по контролируемому сценарию, графики договоров.

 

Контроль заявок по бюджету

 

Если необходимо контролировать заявку по бюджету, то в списке «Настройки программы (БИТ)» необходимо установить флаг:

·         «Контролировать заявку по бюджету» - для контроля заявок на расходование денежных средств по Бюджету движения денежных средств;

·         «Контролировать заявку на затраты по бюджету» - для контроля заявок затраты по Бюджету доходов и расходов.

 

После ввода всех бюджетных данных (запланированных на определенный период сумм оборотов по статьям бюджетирования), необходимо ввести документ «Установка контрольных значений» (меню «Бюджетирование» - «Установка контрольных значений») с видом операции «Контроль заявок».

 

Контроль заявок по бюджету

 

Для установки контроля бюджетов отдельных ЦФО или отдельных операционных бюджетов относительно бюджета, составленного в целом для компании, используется документ «Установка контрольных значений» с видом операции «Контроль бюджета».

 

Контроль осуществляется по сценариям бюджетирования: данные бюджета верхнего уровня (бюджета, составленного в целом для компании) должны быть введены по одному сценарию, а данные бюджетов более низких уровней (бюджеты ЦФО, операционные бюджеты и т.п.)  - по другому сценарию.

 

В форме документа «Установка контрольных значений» в поле Сценарий указывается «контролирующий» сценарий бюджетирования, то есть тот, по которому будут контролироваться данные бюджетов более низких уровней. В поле Сценарий контролируемый указывается сценарий бюджетирования, данные которого будут контролироваться.

 

Для того, чтобы «включить» контроль одного бюджета по данным другого бюджета, необходимо в «контролируемом» сценарии бюджетирования (справочник «Сценарии бюджетирования») на закладке «Настройка контроля» добавить строку, в которой указать дату начала и дату окончания контроля. Если дату окончания не указать, контроль будет продолжаться неограниченное количество времени.

 

 

Контроль графиков договоров по бюджетам

 

Графики платежей и графики начислений по договорам создаются в документе «Дополнительные условия по договору». При проведении документа можно контролировать графики на соответствие бюджетам:  график начислений контролировать по Бюджету доходов и расходов, а график платежей – по  Бюджету движения денежных средств.

Для включения механизма контроля графиков договоров (графиков платежей и графиков начислений) необходимо в списке «Настройки программы (БИТ)» установить флаг «Контролировать графики договоров».

 

Для установки контроля графиков платежей и графиков начислений по бюджетам (БДР и БДДС) используется документ «Установка контрольных значений» (меню «Бюджетирование» - «Установка контрольных значений») с видом операции «Контроль графиков договоров».

 

Для того, чтобы провести документ «Дополнительные условия по договору» сверх бюджета, необходимо установить в документе соответствующий флаг - «Сверх бюджета». Флаг «Не контролировать по бюджету» устанавливается, если какой-то конкретный документ не нужно контролировать по бюджету, например, если графики относятся к периоду, за который еще не введены плановые данные.
