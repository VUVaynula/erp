Документ «Операция (управленческий)»

 

Документ «Операция (управленческий)» предназначен для ввода начальных остатков по счетам управленческого (международного) учета, а также для отражения специфических хозяйственных операций, не имеющих для отражения в учете собственного документа.

 

Начальные остатки вводятся в корреспонденции со вспомогательным счетом 000.

 

Правильность внесения остатков по счетам  можно проверить с помощью стандартного отчета «Оборотно-сальдовая ведомость (управленческий)».
