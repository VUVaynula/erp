Документ «Распределение бюджета»

 

Документ «Распределение бюджета» предназначен для распределения данных бюджетирования (а также любых других данных, имеющихся в информационной базе) по любым базам распределения. Например, можно распределить плановые общехозяйственные расходы на статьи прямых затрат пропорционально фонда заработной платы ЦФО или данные по заработной плате распределить по ЦФО пропорционально штатной численности.

 

В документе указывает сценарий (справочник «Сценарии бюджетирования»), валюта и курс документа.

 

Выбирается настройка распределения из справочника «Настройки распределения бюджета».

 

Настройка распределения создается в справочнике «Настройки распределения бюджета».

 

В настройке указываются:

·         Наименование

·         Вид настройки

o   Простая – данный вид указывается, если в распределении будут участвовать только данные бюджетирования;

o   Источники данных – данный вид указывается, если в распределении будут участвовать не только данные бюджетирования, но и любые другие данные, имеющиеся в информационной базе. При выборе данного вида настройки появляются поля для выбора источников данных для Данных  и Базы (база распределения) из справочника «Источники данных».

 

На закладке «Отборы» указываются отборы для Данных (источник распределения, то есть данные, которые будут распределяться) и Базы (базы распределения, то есть данные, пропорционально которым будет выполнено распределение).

 

На закладке «Настройки» в табличной части «Правила заполнения результата распределения» указываются правила заполнения аналитик в результате распределения. В колонке «Правила заполнения» следует выбрать, какими значениями заполнить соответствующие аналитики в результате распределения.

Для каждой аналитики (Статья оборотов, ЦФО и др.) могут быть следующие варианты источников заполнения:

·         Данные – значения данной аналитики будут браться из Данных (то есть данных, которые будут распределяться),

·         База - значения данной аналитики будут браться из Базы (то есть данных, пропорционально которым выполняется распределение),

·        Указано явно - значения данной аналитики будут браться из явно указанных в табличной части «Правила заполнения результата распределения».

В табличной части «Условия распределения» указывается соответствие аналитик Данных и Базы в случае, если есть различия между ними, например Аналитика 1 (Данные) = Аналитика 3 (База).

 

На закладке «Прочее» указывается комментарий к настройке распределения.

 

После выбора «Настройки распределения» необходимо заполнить табличные части «Данные» и «База». В обеих табличных частях выбирается период, за который будут браться данные. Для автоматического заполнения в соответствии с настройкой распределения следует воспользоваться кнопкой «Заполнить». После заполнения данные могут быть скорректированы вручную.

 

На закладке «Результат распределения» по кнопке «Заполнить и распределить» выполняется автоматическое заполнения и распределение данных (из табличной части «Данные») пропорционально базе распределения (из табличной части «База»). Табличная часть может быть заполнена или скорректирована вручную, в этом случае для выполнения распределения необходимо воспользоваться кнопкой «Распределить».

 

При выполнении распределения может возникнуть ситуация, когда необходимо сторнировать первоначальные данные (из табличной части «Данные»), поскольку после распределения получены те же цифры, только в разрезе других аналитик бюджетирования. На закладке «Прочее» указывается признак сторнирования, если это необходимо.

 

При проведении документа результат распределения отражается в подсистеме бюджетирования.
