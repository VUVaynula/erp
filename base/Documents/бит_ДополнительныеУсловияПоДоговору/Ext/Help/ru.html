Документ «Дополнительные условия по договору»

 

Документ «Дополнительные условия по договору» предназначен для создания и согласования внутри компании графиков платежей (поступлений и выплат денежных средств) и графиков начислений (доходов и расходов).

 

На закладке «Основные» заполняются реквизиты, общие для графиков платежей и графиков начислений: ЦФО, Проект и другие.

 

На закладках «График платежей» и «График начислений» формируются соответствующие графики. В зависимости от вида операции документа и вида договора это может быть график поступлений денежных средств и график доходов или график расходования денежных средств  и график затрат.

 

В параметрах графиков указывается статья оборотов (с типом статьи БДР или БДДС), периодичность (день, месяц и т.п.), период графика, ставка НДС, сумма расхода или прихода. Скрыть параметры графика можно по кнопке  «Параметры».

 

Создание графика происходит по кнопке «График – Создать/Изменить график».

 

В открывшейся форме по умолчанию установлен режим заполнения графика - «Распределение». Возможен также режим «Периодическое заполнение». При «распределении» сумма расхода (прихода) будет равномерно распределена между всеми периодами, входящими а общий период графика. При «периодическом заполнении» сумма расхода (прихода) будет указана в каждом из периодов внутри общего периода графика.

 

В данной форме при необходимости можно изменить период, статью оборотов, сумму и ставку НДС, которые будут использованы при создании графика.

 

По кнопке «Заполнить график – Сформировать график» в табличной части документа сформируется график. По кнопке «Заполнить график – Добавить данные по графику» к имеющимся в табличной части данным добавятся новые данные по графику.

Можно заполнить график платежей по данным графика начислений и наоборот (кнопка «Заполнить»).

 

По кнопке «График – Создать финансовый график» для договоров с типом «Кредиты полученные», «Займы полученные», «Займы выданные» или для договоров с направлением «Финансовый» могут быть автоматически рассчитаны и загружены в документ графики платежей и графики начислений по финансовым договорам. Перед формированием графиков необходимо заполнить график выдачи траншей (на одноименной закладке).

 

По команде в панели навигации «Согласование» происходит согласование документа ответственными лицами компании, которые проставляют свои решения по визам в маршрутах согласования.

 

По кнопке «Создать» в утвержденном документе «Дополнительные условия по договору» в табличных частях «График платежей» и «График начислений» можно создать один или несколько документов планирования: для графика платежей это «Прогнозы платежей», «Заявки на расходование денежных средств» и «Планируемые поступления денежных средств», для графика начислений – «Заявки на затраты». Созданные документы отображаются на закладке «Документы планирования».

 

В табличных частях можно отметить флажками строки графика платежей и графика начислений, по которым необходимо создать документы планирования.

 

Созданные документы планирования можно пометить на удаление на закладке «Документы планирования» (кнопка «Пометить на удаление»).  При этом из формы документа «Дополнительные условия по договору» эти документы будут удалены.

 

Графики платежей и графики начислений при проведении документов «Дополнительные условия по договорам» можно контролировать на соответствие бюджетам – Бюджету движения денежных средств и Бюджету доходов и расходов. Настройка и использование механизма контроля графиков договоров по бюджетам подробно описана в статье «Установка контрольных значений».

Флаг «Сверх бюджета» устанавливается в том случае, если установлен контроль графиков договоров по бюджету (в форме «Настройки программы (БИТ)»), сумма расхода по графикам превышает планируемый расход по бюджету, но график все равно должен быть создан и документ «Дополнительные условия по договору» проведен.

 

Если включен механизм контроля графиков договоров по бюджету, но в каком-то конкретном документе не нужно контролировать графики (например, если графики относятся к периоду, за который еще не введены плановые данные), то в документе «Дополнительные условия по договору необходимо установить флаг «Не контролировать по бюджету».

 

На основании данных графиков платежей и начислений по договорам можно сформировать бюджеты – Бюджет движения денежных средств, Бюджет доходов и расходов. Для этого в документах «Форма ввода бюджета» и «Бюджетная операция» предназначена кнопка «Заполнить - «Копирование данных графиков договоров». При нажатии на эту кнопку открывается форма, в которой можно указать отбор по договору, контрагенту и другим аналитикам, а также конкретный документ «дополнительные условия по договорам», по которому будут заполнены бюджетные данные.
