Форма «Ввод формулы»

 

Форма «Ввод формулы» предназначена для ввода формулы в ячейки макета таких справочников как «Настройки произвольного отчета» и «Элементы данных».

 

 Элементы управления формы:

·        Текст формулы - поле, в котором вводится формула;

·        Панель управления – команды, которые можно использовать в формуле;

·        Таблица параметров – параметры, используемые в формулах;

 

Панель управления

 

На панели управления расположены основные простые команды, используемые в формуле. Также можно перейти к вводу более сложных функций.

 

Описание некоторых команд:

·        «Мин» – ввод функции «Мин»;

·        «Мах» – ввод функции «Мах»;

·        «f()» – открывает форму «Мастер функций» (доступно при вводе формулы из справочника «Настройки произвольного отчета» ;

·        «Коды/имена» – задает режим отображения параметров (доступно при вводе формулы из справочника «Элементы данных») ;

 

Деление рекомендуется задавать через условие, для исключения ошибки деления на ноль. 

Например: ?([Параметр2]=0, 0, [Параметр1]/[Параметр2]).


Кнопка «Перейти к редактору формулы» доступна для ввода формулы из справочника «Настройки произвольного отчета». Позволяет перейти к редактору формул в форме справочника.
