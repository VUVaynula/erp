Обработка «Групповое редактирование дополнительных аналитик»

 

Обработка «Групповое редактирование дополнительных аналитик» (меню «Бюджетирование» - «Групповое редактирование дополнительных аналитик») предназначена для группового заполнения значений в дополнительных аналитиках документов.

 

В форме обработки можно установить отбор: по дате документа, по организации, по настройке, по контрагенту, по выбранному документу и т.д.

 

В табличной части «Типы документов» выбор одного или нескольких документов для проставления дополнительных аналитик.

 

В табличной части «Документы» по кнопке «Обновить» отражаются выбранные документы.

В табличной части «Аналитики документа» по выбранным документам проставляются значения. Возможность заполнения значений: ручной ввод, через заполнение соответствий или заполнение по договору. После заполнения значений, документы проводятся или записываются. По кнопке «Настройка» следует выбрать нужный «Режим сохранения изменений»:

·         режим «Записать аналитики» - сохранение документа.

·         режим «Провести документ» - проведение документа по всем регистрам.

·         режим «Выполнить трансляцию» - выполнение трансляции по настроенным правилам,

режим «Провести по регистрам "БИТ"» - проведение документов по регистрам «Бюджетирование» и «Обороты по бюджету», в этом случае бухгалтерские проводки не перезаписываются.
 
Настройка "Режим записи аналитик":

·        режим «Не перезаписывать существующие». При выборе данного режима заполненные ранее аналитики не будут перезаписаны. При этом можно добавить новую аналитику и установить ее значение;

·        режим «Перезаписывать только установленные». При выборе данного режима перезаписываются только измененные значения;

      ·        режим «Перезаписывать весь набор». При выборе данного режима для каждого выбранного документа перезаписывается весь набор аналитик.
