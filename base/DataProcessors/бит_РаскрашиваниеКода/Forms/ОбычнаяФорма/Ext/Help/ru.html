Обработка "Раскраска кода"

Обработка является примером раскрашивания кода. В данном случае, кода на встроенном языке, но легко может быть дополнена новыми языками.

Форма поделена на две части – два элемент управления: ПолеТекстовогоДокумента и ПолеHTMLДокумента. В первый элемент управления помещается код на встроенном языке платформы 1С:Предприятие, а во второй ЭУ, после нажатия на кнопку "Раскрасить", выводится результат окрашивания. Затем результат можно сохранить в файл.

Исходный код

Текст кода на встроенном языке. Вставляется в элемент управления из буфера или пишется вручную.

Результат

После нажатия на кнопку "Раскрасить" в этом элемент управления будет выведен результат окрашивания кода из ЭУ Исходный код. Результат можно сохранить, нажав на кнопку "Сохранить".
