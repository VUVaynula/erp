Обработка «Назначение заместителей»



Для назначения заместителей по визированию и/или по исполнению Задач при временном отсутствии сотрудника (уход в отпуск, на время болезни и т.п.), предназначена обработка «Назначение заместителей» (раздел «Управление процессами»).


По кнопке «Обновить» в табличной части обработки заполняется список виз и ролей пользователя, на которые у него в системе есть права. Для виз это права на визирование документов в регистре «Права установки виз», для ролей – назначенные роли в регистре «Исполнители задач».

В колонке «Заместитель» выбирается пользователь-заместитель, при этом можно указать по каждой визе или роли отдельного заместителя, либо одного заместителя для всех виз и всех ролей. Для того, чтобы, например, установить заместителя по всем визам пользователя, необходимо выбрать заместителя на уровне группировки «Визы». Если необходимо установить одного заместителя по всем визам и ролям пользователя, то заместитель указывается на верхнем уровне группировки «Пользователь». Для того, чтобы назначить несколько заместителей по одной и той же визе или роли, необходимо скопировать строку с данной визой (ролью) и указать второго заместителя и т.д.

По каждому заместителю (или сразу по всем, в зависимости от уровня группировки) указывается дата начала и дата окончания замещения сотрудника.

В настройках обработки можно установить отборы по пользователям, областям данных (визы или роли) визам, ролям исполнителей.

По кнопке «Записать» вся информация по заместителям фиксируется в специальном регистре «Назначенные заместители».

Назначение заместителей и окончание срока замещения осуществляется с помощью регламентного задания «Обработка назначения заместителей» (раздел «Настройки (БИТ) – команда панели Сервис «Управление регламентными заданиями»). При наступлении даты начала замещения сотрудника в регистрах «Права установки виз» и/или «Исполнители задач» фиксируются соответствующие права и/или назначаются роли. По окончании срока замещения в указанных регистрах с заместителей снимаются назначенные ранее права и роли.



Примечание. В файловом варианте информационной базы для выполнения регламентного задания необходимо перейти в раздел «Администрирование», команда панели навигации «Регламентные и фоновые задания», и запустить отдельный клиентский сеанс.


Назначение сотрудником заместителей для себя доступно только для пользователей с полными правами. Для назначения заместителей любым пользователям информационной базы необходимо установить пользователю дополнительное право администрирования заместителей (раздел «Настройки (БИТ)» - команда панели Сервис «Настройка дополнительных прав пользователей»).
