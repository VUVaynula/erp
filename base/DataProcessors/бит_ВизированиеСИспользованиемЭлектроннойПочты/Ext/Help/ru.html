Обработка «Визирование с использованием электронной почты»

 

Обработка «Визирование с использованием электронной почты» предназначена для настройки оповещений о новых визах с использованием электронной почты.

В данной обработке настраивается:

• учетная запись электронной почты через которую будут выполняться оповещения,
• расписание регламентное задание по взаимодействию,

• состав вложений печатных форм, прикрепляемых к визируемому документу.
