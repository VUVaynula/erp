Обработка «Мои задачи»


Обработка «Мои задачи» служит для отображения списка текущих задач. Открыть обработку можно из раздела «Настройки (БИТ)» и «Управление процессами» форме «Мои задачи».


На закладке «Входящие» отображаются задачи к исполнению, а на закладке «Исходящие» - задачи, по которым пользователь является автором.

Задача может быть принята к исполнению (кнопка «Принять»), выполнена (кнопка «Выполнено») или перенаправлена другому исполнителю (кнопка «Перенаправить»). На закладке «Исходящие» пользователь имеет возможность создавать новые задачи.

Если задача имела ролевую адресацию (т.е. была назначена не лично исполнителю, а роли), то при выполнении команды «Принять» адресация задачи будет изменена и она будет адресована лично текущему пользователю, а не роли. Это удобно использовать для «захвата» задач в том случае, если исполнителями роли являются сразу несколько сотрудников. Также из формы элемента справочника задача может быть отменена автором задачи.

Из формы задачи можно перейти по ссылке к предмету задачи для выполнения требуемых действий с данным объектом системы.

В случае остановки бизнес-процесса, все задачи процесса переходят в состояние «Остановлена». Задачи остановленных бизнес-процессов не отображаются в списке «Мои задачи». После возобновления процесса задачи возвращаются в предыдущее состояние.

Список всех задач можно просмотреть через раздел «Управление процессами» – команда панели навигации «Задачи».

Для удобства просмотра можно установить отбор по автору, по исполнителю, просматривать только выполненные, задачи по определенному процессу или все задачи, а также указать интервал дат по дате записи задачи.

Задачи могут быть созданы на основании основных документов и справочников системы «БИТ.ФИНАНС». При создании задачи «на основании» объект системы становится предметом задачи, и исполнитель может по ссылке перейти к нужному объекту.
