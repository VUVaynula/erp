Обработка «Портал сверки ВГО»

Для сверки данных ВГО компаниями Группы предназначена обработка «Портал сверки ВГО».

Сверка данных ВГО проводится для каждой пары внутригрупповых контрагентов.

В настройках необходимо указать Период сверки и регистр бухгалтерии, по которому выполняется сверка (Журнал проводок (бюджетирование), Журнал проводок (МСФО) и др.).  Так же можно установить отбор по конкретной компании и/или внутригрупповому контрагенту.

На основании заполненных данных пользователь в отдельном окне видит список внутригрупповых контрагентов, для которых либо он, либо внутригрупповой контрагент заполнил документы по сверке ВГО.

В списке для каждой пары компаний Группы отражаются:

·         Акты сверки ВГО, сформированные по отношению друг к другу (только проведенные);

·         Протоколы расхождений, сформированные по отношению друг к другу (только проведенные);

·         Сумма расхождений между компаниями;

·         Подтверждение сверки каждой из компаний (только проведенные).

В рамках данного диалогового окна пользователь имеет возможность создавать новые Акты сверки ВГО и Протоколы расхождений, подтверждать окончание сверки с контрагентом. Для одного внутригруппового контрагента в рамках одного периода может быть проведен в системе только один Акт сверки ВГО и несколько Протоколов расхождений. В системе поддерживается версионирование данных, то есть Акты сверки ВГО и Протоколы расхождений могут иметь несколько версий, из которых пользователь может выбрать актуальную версию.

После того, как для внутригруппового контрагента были загружены данные двумя сторонами, пользователь может построить по данному контрагенту отчет о статусе сверки.

Сверка в рамках периода по каждой паре компаний ведется, пока уровень расхождений будет меньше установленного допустимого уровня. Предельно допустимый уровень расхождений устанавливается в Настройках проведения сверки ВГО.
