Обработка «Пакетное создание форм ввода бюджета»

Обработка «Пакетное создание форм ввода бюджета» предназначена для автоматического создания документов «Форма ввода бюджета» по выбранным Настройкам форм ввода бюджета. Обработка позволяет так же выгрузить из созданных документов шаблоны MS Excel в выбранный каталог, и после заполнения их центрами финансовой ответственности загрузить обратно в документы «Форма ввода бюджета».

 В обработке в шапке заполняются общие для всех документов реквизиты: Период, Сценарий, Валюта и Курс, Режим формирования зависимых оборотов и Режим формирования движений (для Настроек форм ввода с фиксированным макетом). В табличной части выбираются Настройки форм ввода бюджета, заполняются необходимые реквизиты. Для каждого документа можно указать  свой ЦФО, Проект, другие аналитики, Ответственного и Комментарий. По команде «Создать документы» создаются документы «Форма ввода бюджета» по выбранным настройкам.

По кнопке «Удалить документы» можно удалить созданные документы из базы, если они были созданы ошибочно.

По команде «Создать задачи» можно создать задачи всем ответственным, при этом в предмет задачи будет записана ссылка на созданный документ «Форма ввода бюджета».

На закладке «Выгрузка в Excel» указывается каталог, в который будут выгружены шаблоны Excel для заполнения их ответственными за бюджеты сотрудниками. По кнопке «Выгрузить шаблоны в Excel» осуществляется создание файлов и сохранение их в выбранном каталоге. Имя файлов формируется автоматически и показывается в соответствующей колонке.

 Для загрузки заполненных файлов Excel необходимо на закладке «Загрузка из Excel» указать каталог, выбрать файлы для загрузки и нажать кнопку «Загрузить из Excel».
