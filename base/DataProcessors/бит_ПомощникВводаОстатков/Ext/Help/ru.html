Обработка «Помощник ввода остатков (БИТ)»

Обработка «Помощник ввода остатков (БИТ) позволяет групповым образом ввести начальные остатки по правилам трансляции. В результате формируются документы «Операция (управленческий)» по выбранному регистру.

Так же для подсистемы МСФО в данной обработке имеется возможность групповым образом сформировать документы «Принятие к учету ОС (МСФО)» и «Принятие к учету НМА (МСФО)», заполнив их на основании данных российского бухгалтерского учета.

В шапке обработки заполняются следующие основные настройки:

  • ·         Организация – организация, по которой выполняется ввод начальных остатков (элемент справочника «Организации»);
  • ·         Дата ввода остатков – дата, которой будут сформированы документы по вводу начальных остатков;
  • ·         Регистр приемник – регистр, по которому выполняется ввод начальных остатков;
  • ·         Регистр источник – регистр, на основании данных которого выполняется ввод начальных остатков;
  • ·         Вспомогательный счет – счет, в корреспонденции с которым, формируются проводки по вводу начальных остатков;
  • ·         Показатели – ресурсы регистра-приемника, которые необходимо отображать в обработке.

Счета плана счетов, соответствующие регистру – приемнику заполняются на закладках «Основные счета» и «Забалансовые счета», в зависимости от вида счета.

Колонка «Остатки регистра-приемника»

Суммовые показатели в колонке «Остатки регистра-приемника» заполняются при нажатии на кнопку «Остатки» остатками из регистра-приемника. Данная информация является справочной.

Колонка «Остатки регистра-источника»

Суммовые показатели в колонке «Остатки регистра-источника» заполняются при нажатии на кнопку «Заполнить через трансляцию».

Для удобства работы с большим массивом данных, пустые строки в табличных частях можно скрыть. Для этого предназначен параметр «Скрыть пустые строки» в настройках обработки (Кнопка «Настройки»):

Создание документов по вводу остатков

Для создания документов по вводу начальных остатков необходимо с помощью кнопки «Заполнить через трансляцию» заполнить колонку «Остатками регистра-источника», выделить флагами те строки, по которым требуются создать документы и нажать на кнопку «Создать документы».
