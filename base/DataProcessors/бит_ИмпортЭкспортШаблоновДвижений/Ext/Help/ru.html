Обработка «Импорт/экспорт шаблонов движений»

 

Обработка «Импорт/экспорт шаблонов движений» универсальная обработка предназначена  для обмена настроенными шаблонами и комплектами движений между информационными базами. С помощью данной обработки можно обмениваться только шаблонами движений и комплектами движений.


Форма элемента имеет две закладки:


  • Экспорт;
  • Импорт.


На закладке «Экспорт» в поле файл экспорта указывается наименование файла и адрес выгрузки комплектов шаблонов движений. Также на закладке представлены два поля для подбора комплектов и шаблонов, подлежащих экспорту.


На закладке «Импорт» в поле файл импорта указывается наименование файла, импортируемого в систему. Ниже отображается протокол загрузки импортируемого файла.
