Редактирование выражений.

Доступно для колонок с режимом поиска «Вычислять».


В переменную «Результат» необходимо поместить результат выражения.

Можно получить данные строки через «ТекущиеДанные». Но только для тех данных, которые находятся в строке перед ячейкой, для которой вычисляется выражение.

Текст текущей ячейки находится в переменной «ТекущаяЯчейка».

Можно получить текст каждой ячейки строки через массив «ТекущиеЯчейки». Например, «ТекстыЯчеек[4]», где 4 - номер в массиве (соответсвует номеру колонки). Текст в ячейке не является значением.


Пример вычисления основного банковского счета по контрагенту:

Если ЗначениеЗаполнено(ТекущиеДанные.Контрагент) Тогда

Результат = ОбщегоНазначения.ЗначениеРеквизитаОбъекта(ТекущиеДанные.Контрагент, "ОсновнойБанковскийСчет");

КонецЕсли;

Будет работать только если поле контрагента в строке табличного документа стоит перед полем банковского счета.
