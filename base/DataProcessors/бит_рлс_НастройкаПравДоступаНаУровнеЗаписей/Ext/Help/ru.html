Обработка «Настройка прав доступа на уровне записей»

 

Настройка прав доступа может быть выполнена с помощью обработки «Настройка прав доступа на уровне записей» (меню «Настройки (БИТ)» - «Права доступа» - «Настройка прав доступа на уровне записей»).

 

В шапке указывается Пользователь или Группа пользователей, для которых настраиваются права доступа к данным.

 

На закладках «ЦФО и Статьи оборотов», «Проекты» и «Исполнители» описываются права доступа к данным отдельно по каждому разделителю.

 

В поле «Отбор» указываются условия отбора для значений областей доступа.

 

В табличной части «Настройки прав доступа» с помощью установки флажков настраиваются права на чтение и запись (редактирование) данных. По кнопке «Обновить» табличная часть заполняется списком значений областей доступа. Реквизит Пользователь/Группа по умолчанию заполняется тем же значением, что и в шапке формы обработки, но можно указать и другое значение. Вид настройки может принимать значения Доступ или Запрет.

 

На закладке «ЦФО и Статьи оборотов» флажки устанавливаются для пары ЦФО - Статьи оборотов, на закладках «Проекты» и «Исполнители» – для конкретного значения области доступа.

 

Для установки прав доступа необходимо нажать кнопку «Установить».
