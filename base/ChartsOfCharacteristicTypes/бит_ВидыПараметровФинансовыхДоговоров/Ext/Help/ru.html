План видов характеристик «Виды параметров финансовых договоров»

 

Предназначен для хранения возможных параметров финансовых договоров, например, «Периодичность», «Дата первого платежа», «Алгоритм расчета» и др.

 

В плане видов характеристик присутствуют предопределенные виды параметров, которые используются в системе в алгоритмах расчета графиков платежей по финансовым договорам:

1.      Алгоритм расчета - параметр предназначен для указания алгоритма расчета суммы для каждой составляющей платежа;

2.      Дата первого платежа - параметр предназначен для указания даты первого платежа отдельно для каждой составляющей платежа;

3.      Процентная ставка - параметр предназначен для указания значения процентной ставки по договору, а также ставок комиссий;

4.      Тип процентной ставки - может принимать значения «Фиксированная» или «Плавающая»;

5.      Периодичность - периодичность амортизации основного долга, начисления процентов и комиссий;

6.      Статья оборотов БДДС (поступление) - статья оборотов для планирования поступлений кредитных средств от банка или поступлений сумм по возврату выданного займа;

7.      Статья оборотов БДДС (расходование) - статья оборотов для планирования платежей по основному долгу, процентам, комиссиям;

8.      Статья оборотов БДР (поступление) - статья оборотов для планирования доходов от процентов по займам выданным;

9.      Статья оборотов БДР (расходование) - статья оборотов для планирования расходов по кредитным договорам.

 

В план видов характеристик могут быть добавлены новые виды параметров, но для того, чтобы использовать их при расчетах графиков, необходимо будет написать новые алгоритмы расчетов в справочнике «Пользовательские функции».

 

Флаг «Обязательный» устанавливается для параметров, обязательных для заполнения. Если флаг установлен, то график платежей или график начислений не будет сформирован, если не будет заполнено значение данного параметра.
