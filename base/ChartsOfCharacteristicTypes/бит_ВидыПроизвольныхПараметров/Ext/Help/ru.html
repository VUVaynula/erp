«Виды произвольных параметров»

План видов характеристик «Виды произвольных параметров» (раздел «Настройки (БИТ)» - подраздел панели навигации «Произвольные параметры») предназначен для задания виды параметров, которые планируется использовать. Для каждого вида параметра указывается наименование, тип параметра (Число, Строка, Дата, Булево), единица измерения (выбирается из справочника «Валюты» (для суммовых показателей) или из классификатора единиц измерения).
