План видов характеристик «Настройки программы»

 

В конфигурации заданы предопределенные значения настроек конфигурации.

Список предопределенных значений:

  • Допустимый % превышения суммы заявки
  • Запретить платежи без заявки
  • Контролировать заявку по бюджету
  • Контролировать остатки по общей заявке
  • Не формировать движения по факту автоматически
  • Фильтровать статьи оборотов по принадлежности

 Для хранения настроек предназначен регистр сведений «Настройки программы (БИТ)».
