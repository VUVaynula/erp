Регистр сведений «Объекты мобильного визирования»


Представление объектов визирования:


В версии мобильного приложения (2.1.3) и конфигурации БИТ.ФИНАНС для 1С:Бухгалтерия предприятия – не ниже 3.1.31.2, есть возможность настраивать представления объектов. Настройки происходит на стороне информационной базе, в режиме 1С:Предприятия, с помощью регистра сведений «Объекты мобильного визирования».


В поле синоним указывается шаблон для объекта системы, удобнее это делать помощью кнопки «Вставить шаблон».


Поля выбираются из тех, что присутствуют на форме мобильного приложения. При выборе полей с типа Дата и Сумма, к ним автоматически добавляется формат после “:”, который можно изменить или удалить при необходимости.


После указания синонимов для объектов, в мобильном приложении необходимо обновить настройки визирования с помощью кнопки «Получить». При успешном получении настроек, будет выдано соответствующее сообщение.


Также в форме самого объекта будет виден и доступен для редактирования синоним, заданный в информационной базе.


После успешной настройки, список объектов для согласования станет гораздо понятнее пользователям.


Добавление новых документов для согласования:


Если в информационной базе есть объекты, к которым добавлены механизмы согласования БИТ.ФИНАНС (с помощью доработок конфигурации), то данные документы также можно будет согласовывать с помощью мобильного приложения. Для этого их необходимо добавить в регистр сведений – Объекты мобильного визирования.
