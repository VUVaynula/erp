Регистр сведений «Соответствие операций ВГО»


Регистр предназначен для настройки соответствия групп операций (и/или отдельных операций) «поставщика» и «покупателя». Например, группа операций «Реализация товаров, работ, услуг» будет соответствовать группе операций «Поступление товаров, работ, услуг».


Данное соответствие (групп операций ВГО) используется при построении отчета «Статус сверки ВГО», который служит для проведения сверки между двумя компаниями Группы. Если соответствие установлено между группами операций ВГО, то в отчете выводится расхождение между внутригрупповыми оборотами двух контрагентов именно по группе операций, а не по операциям, входящим в группу.
