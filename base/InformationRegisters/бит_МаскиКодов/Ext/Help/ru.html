Регистр сведений «Маски кодов»


Регистр предназначен для установки маски кодов планов счетов.


Максимально возможная длина маски кода составляет 15 символов. В поле «Маска кода» устанавливается маска, которая будет использоваться в выбранном плане счетов.


При указании маски кода значимые символы необходимо заменять символом @. В качестве разделителя возможно использовать один из трех символов: ".", "/", "-".


Пример маски кода, состоящей из трех групп по три цифры каждая с точкой в качестве разделителя, будет выглядеть следующим образом: @@@.@@@.@@@


После установки маски кода необходимо сконвертировать коды выбранного плана счетов. Для этого служит кнопка «Конвертировать коды».
