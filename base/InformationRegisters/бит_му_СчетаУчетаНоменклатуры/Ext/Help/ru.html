Регистр сведений «Счета учета номенклатуры»


Регистр предназначен для хранения сведений о типовых счетах учета номенклатуры для целей МСФО, с целью автоматической подстановки счетов учета в соответствующие реквизиты документов при выборе номенклатуры.


Могут быть указаны счета учета и счета учета резерва обесценения:

  • Для конкретной номенклатуры (эти данные являются наиболее приоритетными при автозаполнении счета);
  • Для номенклатурной группы (эти данные подставляются по тем элементам номенклатуры, которые входят в определенную номенклатурную группу, но по которым не указаны сведения для номенклатуры);
  • Для склада;
  • Для организации.


Приоритетным является конкретное соответствующее поле условия перед условиями «по умолчанию» (с неуказанными значениями полей). Если соответствуют несколько полей, то приоритет убывает в последовательности: номенклатура, номенклатурная группа, склад, организация. То есть, если, например, для склада Основной задан один счет учета (вне зависимости от организаций), а для организации Фирма задан другой счет учета (вне зависимости от складов), и не заданы прочие условия, то в случае учета МПЗ, принадлежащего Фирме, на складе Основной подставится счет учета, указанный для склада.
