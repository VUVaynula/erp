Регистр сведений «Настройки расчета отложенных налогов»


Регистр предназначен для хранения информация, из каких источников на определенную дату собираются данные для расчета отложенных налогов по видам активов и обязательств. Для одного и того же вида активов или обязательств может использоваться более одного источника данных.


Вносить данные в регистр можно непосредственно вручную или с помощью специализированной обработки «Настройки расчета отложенных налогов». Использование обработки может быть удобнее тем, что она представляет «срез» настроек на определенную дату, в том числе показывает те виды активов и обязательств, по которым настроек на заданную дату нет (в табличной части такие строки оттенены серым шрифтом).


Флаг «Выключить общие» означает, что для конкретной организации применяется особый порядок расчета отложенных налогов (т.е. настройки действуют только применительно к данной организации).


По кнопке «Записать» сведения записываются в регистр.
