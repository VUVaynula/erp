Регистр сведений «Назначение дополнительных аналитик»


Регистр предназначен для назначения аналитик бюджетирования бухгалтерским документам.


В дальнейшем эти данные используются для отражения фактических данных по движению денежных средств, при проведении платежных документов, в разрезе нужных аналитик бюджетирования, а также при трансляции данных.


В регистре «Назначение дополнительных аналитик» бухгалтерским документам можно назначить свой набор аналитик бюджетирования, которые будут отображены в документах на закладке «Дополнительные аналитики» при нажатии кнопки «Заполнить».


Значения аналитик могут быть заполнены автоматически исходя из установленных соответствий в регистрах «Соответствие статей оборотов и статей регламентированного учета», «Соответствие ЦФО и подразделений», «Соответствие статей оборотов и номенклатурных групп», а также исходя из Настроек пользователя (БИТ). В Константах (БИТ) на закладке «Прочее» можно выбрать один из режимов автозаполнения аналитик.

Режимы:

  • Не заполнять – в документах бухгалтерского учета не будет заполняться перечень аналитик и их значения;
  • Заполнять всегда – в документах бухгалтерского учета всегда будет заполняться список аналитик из регистра «Назначение дополнительных аналитик» и значения этих аналитик по соответствию из указанных выше регистров. Если бухгалтерские документы ранее были проведены без заполнения дополнительных аналитик, то после перепроведения документов аналитики будут заполнены;
  • Заполнять только для новых документов – то же что для режима «Заполнять всегда», но только для новых документов.


После заполнения аналитик бюджетирования нужно заполнить их значения и провести документ. При проведении документов могут быть сформированы фактические данные по движению денежных средств в разрезе указанных аналитик, а также данные по Бюджету доходов и расходов и проводки по счетам плана счетов бюджетирования, МСФО и другим планам счетов.
