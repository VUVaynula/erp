Регистр сведений «Начало открытого периода»


Регистр предназначен для хранения данных о дате начала открытого периода. Установление даты открытого периода происходит в разрезе каждой организации.


Используется в подсистеме визирования.


Указываются параметры:

  • Организация – юридическая единица, по которой ведется учет;
  • Объект системы – регистр бухгалтерии, по которому необходимо контролировать закрытый период;
  • Дата открытого периода - дата начала открытого финансового периода, начиная с которой движения формируются без сторнирующих проводок.
