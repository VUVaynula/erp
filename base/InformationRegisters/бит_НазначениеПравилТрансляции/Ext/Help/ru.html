Регистр сведений «Назначение правил трансляции»


Регистр предназначен для назначения правил трансляции организации или группе организаций.


Описание реквизитов:

  • Регистратор – документ «Назначение правил трансляции», который актуализирует правила трансляции для структурной единицы (организация или группа организаций);
  • Номер строки – номер строки в табличной части документа-регистратора «Назначение правил трансляции»;
  • Активность – флаг проставляется, если данное правило распространяется на структурную единицу;
  • Дата начала – календарная дата, начиная с которой правило становится актуальным для структурной единицы;
  • Структурное подразделение – организация, на которую распространяется данное правило;
  • Дата окончания – календарная дата, начиная с которой правило становится не актуальным для структурной единицы;
  • Приоритет – степень значимости правила.
