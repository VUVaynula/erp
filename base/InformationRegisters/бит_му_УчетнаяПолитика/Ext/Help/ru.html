Регистр сведений «Учетная политика (МСФО)»


Регистр предназначен для хранения данных об учетной политике организации для международного учета.


Каждая запись в регистре характеризует учетную политику организации за определенный период.


Пользователь фиксирует дату начала действия параметра, организацию, на которую распространяется это правило, а также настройку и значения.


Для регистра «Учетная политика (МСФО)» реализован контроль удаления и изменения параметров. Если сформировано хотя бы одно движение с участием данного параметра, то удаление и изменение строки регистра сведений невозможно.


Параметры учетной политики, устанавливаемые пользователем:

  • Валюта международного учета - функциональная валюта международного учета, в которой формируются движения по регистру бухгалтерии "Международный";
  • Флаг «Начислять амортизацию с текущего периода» - согласно ПБУ 06/01 начисление амортизации происходит с первого числа месяца, следующего за месяцем принятия этого объекта к учету. При активации флага, начисление амортизации начнется с месяца принятия объекта к учету по МСФО. При деактивации флага, начисления начнутся со следующего месяца;
  • Счет нераспределенной прибыли - счет плана счетов международного учета отражающий нераспределенную прибыль (убыток) компании;
  • Метод учета амортизации при переоценке - при учете по модели переоцененной (справедливой стоимости) компания может выбрать один из методов учета амортизации: пропорциональная переоценка суммы амортизации или списание амортизации.
