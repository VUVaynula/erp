Регистр накопления «Обороты по типам проводок (БИТ)»


Регистр предназначен для хранения оборотов в разрезе типов проводок.


Записи в данный регистр формируются параллельно при трансляции из регистра бухгалтерии«Журнал проводок (бухгалтерский и налоговый учет)» в регистр бухгалтерии«Журнал проводок (международный)» в том случае если установлен параметр «Вести учет по типам проводок» в «Константы (БИТ)», закладка «МСФО».


На основание данных данного регистра формируется отчет «Отчет по типам проводок (МСФО)».
