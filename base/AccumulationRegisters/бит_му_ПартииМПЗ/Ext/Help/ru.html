Регистр накопления «Партии МПЗ»


Регистр предназначен для накопления информации о движении партий МПЗ на складах компании, в том числе закупленных у собственных контрагентов.


Измерения:

  • Организация (организация, чьи товародвижения регистрируются);
  • Склад (склад организации);
  • Номенклатура (МПЗ);
  • Документ оприходования (партия товара - документ, с помощью которого был оприходован товар);
  • Характеристика номенклатуры;
  • Серия номенклатуры;
  • Качество;
  • Счет учета (счет учета номенклатуры (МУ);
  • Партия собственного контрагента (документ, с помощью которого партия товара была оприходована поставщиком-собственным контрагентом).

Ресурсы:

  • Сумма упр. (себестоимость товара);
  • Сумма поступления упр. (себестоимость товара у собственного контрагента, на момент реализации товара);
  • Количество (количество данных номенклатурных позиций в партии).
