Справочник «Периметры консолидации»


Справочник предназначен для хранения периметров консолидации – перечень компаний включенных в группу, с целью  формирования консолидированной отчетности. В качестве элементов периметра могут выступать как реальные, так и виртуальные (реально не существующие, но необходимые для определенных целей) организации.


Количество элементов справочника не ограничено и определено организационной структурой холдинга


В шапке элемента:


  • Группа (элементы справочника можно объединять в группы);
  • Наименование (текстовое поле);
  • Период (дата, с которой определяется состав периметра).


Табличное поле содержит следующие колонки:


  • Организация (элемент справочника «Организации»);
  • Дата окончания (тип «Дата», дата выхода организации из периметра);
  • Тип консолидации (организационные единицы подразделяются на обычные и консолидирующие (фиктивные), для отражения элиминационных и консолидационных корректировок);
  • Полная доля владения, % (доля владения организационной единицы для конкретного периметра, с учетом прямых и косвенных долей владения, измеряется в процентах);
  • Доля неконтролирующих акционеров, % (доля неконтролирующих акционеров рассчитывается как разница 100% и полной доли владения, измеряется в процентах);
  • Метод консолидации (Полная консолидация, Долевой метод, Пропорциональная консолидация);
  • Флаг «Осуществляет контроль периметра» (активируется для материнской компании, используется в других объектах модуля для корректного расчета нераспределенной прибыли и т.д.).


Данные табличного поля можно по кнопке «Записать» записать в регистр сведений «Состав периметров консолидации». По кнопке «Записать и закрыть» элемент справочника будет записан в регистр сведений «Состав периметров организации» и закрыта форма редактирования.
