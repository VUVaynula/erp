Справочник «Бюджеты»

 

Справочник «Бюджеты» предназначен для хранения списка бюджетов, которые планируется составлять в организации (бюджет доходов и расходов, бюджет движения денежных средств и т.д.), и их структуры. Для каждого бюджета задается свой набор статей оборотов, в разрезе которых будет составляться бюджет и формироваться отчеты.

В форме элемента справочника «Бюджеты» заполняются следующие реквизиты:

В шапке документа:

·        Наименование – наименование бюджета (например, «БДДС», «БДР» и т.п.);

·        Полное наименование – полное наименование бюджета (например, «Бюджет доходов и расходов»);

·        Тип бюджета – реквизит может принимать значения: БДР (бюджет доходов и расходов), БДДС (бюджет движения денежных средств), Прочие (прочие бюджеты);

 

В табличной части «Статьи бюджета» создается структура бюджета – из справочника «Статьи оборотов» подбираются необходимые статьи оборотов. Для заполнения списка статей оборотов можно воспользоваться кнопкой «Подбор». Для каждой статьи оборотов указывается:

·        Наименование – краткое наименование статьи оборотов для данного бюджета;

·        Полное наименование – полное наименование статьи оборотов для отображения в отчетах по бюджету;

·        Статья оборотов – статья оборотов, значение оборота по которой будет выведено в строку бюджета,  элемент справочника «Статьи оборотов»;

·        Коэффициент – коэффициент пересчета суммы оборота по данной статье, полученный результат отображается в отчете по бюджету;

·        Кодификатор – произвольный код статьи оборотов, используемый для удобства работы со справочником, а также для вывода в печатных формах и отчетах;

·        Не отображать – если флаг установлен, то данная статья не будет отображаться в отчетах по бюджету;

·        Комментарий – произвольный комментарий к бюджету.

 

В табличной части «Счета бюджета» можно указать счета плана счетов для бюджетирования для вывода начального и конечного остатков по этим счетам в отчете «Отчет по бюджету».
