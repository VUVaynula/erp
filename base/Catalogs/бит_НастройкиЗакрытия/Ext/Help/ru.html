Справочник «Настройки закрытия»

 

 1.      Назначение справочника

 

 Справочник «Настройки закрытия» предназначен для формирования и хранения правил закрытия счетов. Эти настройки используются в документе «Закрытие счетов» при заполнении.  В сформированном элементе справочника отражаются данные, которые подлежат закрытию, вид закрытия, счет закрытия и база для распределения.

 

 2.      Описание реквизитов

 

В шапке элемента справочника «Настройки закрытия» заполняются следующие реквизиты:

  • Наименование – наименование настройки закрытия (например, «Директ-костинг», «Распределение по нескольким счетам» и т.п.);
  • Регистр бухгалтерии - элемент списка регистров бухгалтерии («Журнал проводок (бухгалтерский учет)»).

 

Форма элемента справочника разделена на две закладки: «Алгоритм закрытия» и «Дополнительно».

 

Для каждой настройки необходимо указать ряд параметров. Эти данные отражаются в табличной части на закладке «Алгоритм закрытия».

 

Вид закрытия – параметр определяет принцип закрытия счета: простое или распределение.  Если указано в поле значение «Простое», то вся сумма счета, подлежащего закрытию, будет перенесена на новый счет. Если указано в поле значение «Распределение», то сумма с закрываемого счета будет разбита пропорционально базе распределения;

 

Данные – элемент справочника «Источники данных», определяющий параметры отбора сумм счета или счетов, подлежащих  закрытию;

 

Счет закрытия – счет, на который переносится сумма подлежащая закрытию. Заполнение поля необходимо только в том случае, если счет закрытия один.  Если распределение происходит на группу счетов, то данное поле остается пустым;

 

Сторона закрытия – поле заполняется значением: дебет  или кредит, если счет, подлежащий закрытию, является активно-пассивным;

 

Аналитика счета закрытия – в данной колонке отражаются виды субконто, соответствующие счету закрытия. Можно явно задать в табличной части аналитику, подставляемую в проводку. Движения формируются документом «Закрытие счетов»;

 

База – элемент справочника «Источники данных». Основа для пропорционального распределения суммы со счета, подлежащего закрытию. Поле заполняется, если  указан вид закрытия «Распределение». В качестве базы распределения может выступать счет, группа счетов, счет ограниченный значениями аналитики;

 

Поле базы –  параметр для распределения, в качестве которого выступает субконто счета закрытия;

 

Поле данных –  параметр детализации распределения счета, подлежащего закрытию;

 

Заполнение субконто –  вид аналитики счета закрытия, проставляемой в проводке. Движение формируется документом «Закрытие счетов». В поле «Заполнение субконто» может быть выбрана:


  • аналитика счета, используемого в качестве базы распределения (например, База Субконто 1);
  • аналитика счета, подлежащего закрытию (например, Данные Субконто 1);
  • аналитика счета закрытия (например, ДанныеКорСубконто 1);
  • аналитика задана явно (Указано явно);

 

Имя количество – имя реквизита;

 

Имя валюта – имя реквизита;

 

Имя вал. сумма – имя реквизита;

 

Имя сумма – имя реквизита (например, Сумма);

 

Содержание – описание движения, формируемого документом «Закрытие счетов». Текст поля отражается в проводке в одноименном поле «Содержание»;

 

№ журнала – данные поля отражаются в проводке, формируемой документом «Закрытие счетов».

 

На закладке «Дополнительно» отражена табличная часть «Соответствие имен ресурсов», в которой указываются имена ресурсов и реквизитов.
