Справочник «Шаблоны прав доступа»

 

Справочник «Шаблоны прав доступа» является вспомогательным для настройки прав доступа.

 

Алгоритм использования:

  • Сначала с помощью обработки «Настройка прав доступа на уровне базы» настраиваются права для шаблона;
  • Потом права пользователей заполняются по настроенному шаблону с помощью обработки.
