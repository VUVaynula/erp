Справочник «Настройки актов сверки ВГО»

 

Справочник «Настройки акта сверки ВГО» предназначен для хранения возможных настроек Актов сверки ВГО, формируемых в системе.

 

Настройка акта сверки ВГО позволяет определить основные параметры актов сверки, которые будут формироваться компаниями Группы:

·         Счета расчетов

·         Корреспондирующие счета по операциям, кроме оплаты

·         Корреспондирующие счета оплаты

·         Субконто счетов

·         Регистр бухгалтерии, по которому будет проводиться сверка

 

Счета добавляются по кнопке «Добавить» или Insert. Если добавить счет первого уровня, например 62 и установить курсор на этой строке, то по кнопке «Загрузить подчиненные» в табличную часть загрузятся все подчиненные субсчета данного счета.

 

Для каждого счета необходимо определить признак является ли этот счет счетом расчетов («Счет расч.»), счетом оплаты («Счет опл.») или прочим корреспондирующим счетом оборотов («Счет об.»). По кнопке «Изменение признаков» можно устанавливать или изменять эти признаки для выделенных счетов (выделение группы счетов -  с кнопкой Shift).

 

На закладке «Дополнительно» может быть установлен флаг «Разрешено выбирать операцию ВГО». В этом случае можно будет заполнять акт сверки вручную.
