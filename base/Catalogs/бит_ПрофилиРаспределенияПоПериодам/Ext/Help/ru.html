Справочник «Профили распределения по периодам»

 

Суть механизма распределения оборотов по периодам состоит в том, что при установке зависимостей оборотов по статьям с помощью документа «Установка зависимых оборотов» , можно указать сдвиг во времени, с которым зависимая статья будет отстоять от исходной, либо задать распределение суммы зависимой статьи по нескольким периодам.

Для настройки таких зависимостей необходимо настроить соответствующие профили в справочнике «Профили распределения по периодам».

 

При настройке номера периода в колонке «Номер периода» текущий месяц обозначается цифрой 0, следующий за текущим – цифрой 1, следующий за этим месяцем – цифрой 2 и т.д. Предыдущий месяц обозначается цифрой -1, более ранний – цифрой -2 и т.д. В колонке «Процент (%)» указывается, сколько процентов от суммы оборота по основной статье должна составлять сумма оборота по зависимой статье в данном периоде. Общая сумма процентов должна быть равна 100%.
