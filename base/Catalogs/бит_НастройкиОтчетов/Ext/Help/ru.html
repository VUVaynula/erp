Справочник «Настройки отчетов»

 

Справочник «Настройки отчетов» предназначен для хранения настроек вывода «Универсального отчета по источникам данных». 

 

 Для  каждого элемента справочника указываются:

·        Наименование – произвольное наименование отчета;

·        Группа – группа справочника «Настройки отчетов»;

·        Источник данных -  может принимать значение из справочника «Источники данных» или  из справочника «Способы компоновки источников данных».

·        Схема компоновки – используется для настройки вывода данных отчета. Схему компоновки можно заполнить автоматически и настроить вручную.  В схеме компоновки можно  задать оформление, порядок вывода, группировки и т. д.

 

По кнопке «Сформировать отчет» можно проверить настройки отчета.
