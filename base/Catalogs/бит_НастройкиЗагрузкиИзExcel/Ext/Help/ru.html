Справочник «Настройки загрузки из Excel»

 

Справочник «Настройки загрузки из Excel» предназначен для хранения настроек загрузок из файлов Excel.

 

 В справочнике указываются:

·        Настройки чтения из Excel – настройки по умолчанию;

·        Правила загрузки – устанавливаются правила загрузки из Excel: настраивается соответствие между реквизитами справочника и реквизитами файла загрузки. По кнопке «Выгрузить структуру» можно автоматически выгрузить в файл Excel структуру реквизитов файла;

·        Ключевые поля  - поля поиска по реквизитам загружаемых объектов. При установленном флаге  «код» и «наименование» поиск будет происходить по совокупности двух реквизитов.

·        Значения по умолчанию – устанавливаются для реквизитов значение которых необходимо заполнить фиксированным значением указанном в колонке «По умолчанию». Если необходимо заполнить фиксированное значение только для пустых значений, тогда необходимо установить флаг «Только для пустых».
