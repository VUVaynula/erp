Справочник «Операции ВГО»

 

Справочник «Операции ВГО» предназначен для хранения списка возможных хозяйственных операций, по  которым проводится выверка внутригрупповых оборотов. Справочник иерархический.

 

Операции, а также группы операций делятся на следующие виды:

·         Операции поставщика и операции покупателя. Данные операции делятся на следующие типы:

o   Операции начисления задолженности (например «Приобретение ОС», «Реализация ОС»)

o   Операции погашения задолженности (например «Поступления от реализации товаров, работ услуг», «Погашение задолженности перед поставщиками»);

·         Технические операции по переносу оборотов между счетами расчетов (например «Зачет аванса»).

 

Вид и тип операции указывается не только для элемента справочника, но и для группы операций.
