Справочник «Настройки произвольных отчетов»

 

Справочник «Настройки произвольных отчетов» предназначен для хранения макета и настроек вывода «Произвольного отчета».

 

 В настройках произвольного отчета задается:

·        Наименование - произвольное наименование отчета;

·        Группа – группа справочника «Настройки произвольных отчетов»;

·        Способ компоновки – элемент из справочника способы компоновки источников данных;

·        Формат – формат отображения чисел в отчете;

·        Макет – фиксированный макет произвольного отчета;

·        Прочее – описание отчета.

 

Описание заполнения макета

 

В таблице макета пользователь задает макет и для вычисляемых ячеек указывает правила заполнения ячеек. Для того чтобы задать правило для текущей ячейки необходимо открыть окно установки правила. Окно открывается по кнопке «Действия макета» - «Установить правило заполнения области», при нажатии сочетания клавиш «Ctrl+Space» или при добавлении строки в окне «Правила заполнения».  Видимость окна «Текущие правила заполнения» управляется кнопкой «Правила заполнения».

 

В окне  «Правило заполнения области» пользователь задает:

·        Имя области – адрес ячейки;

·        Вид ячейки – может принимать одно из значений «Значение» или «Формула»;

·        Элемент данных – элемент данных текущего способа компоновки данных. Реквизит доступен для вида ячейки «Значение»;

·        Имя ресурса – имя ресурсов текущего способа компоновки данных. Реквизит доступен для вида ячейки «Значение»;

·        Формула -  формула расчета по вычисляемым ячейкам макета. Реквизит доступен для вида ячейки «Формула». Формула задается в окне ввода формулы.

 

По кнопке «Действия макета» - «Раскрасить ячейки» вычисляемым ячейкам устанавливается цвет по виду ячейки. Для ячеек с видом «Значение» устанавливается желтый цвет. Для ячеек с видом «Формула» устанавливается зеленый цвет.
