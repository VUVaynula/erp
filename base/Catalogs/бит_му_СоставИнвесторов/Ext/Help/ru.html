Справочник «Состав инвесторов»


Справочник предназначен для определения состава инвесторов организационной единицы.  В качестве инвесторов могут признаваться элементы справочника «Организации», элементы справочника «Физические лица», а также предопределенный элемент «Неконтролирующие акционеры».  


В шапке элемента:


·        Группа (элементы справочника можно объединять в группы);

·        Наименование (текстовое поле);

·        Объект инвестирования (элемент справочника «Организации»);

·        Период (дата, с которой определяется состав инвесторов).


Если наименование пользователем не установлено, то по умолчанию проставляется наименование объекта инвестирования.


Табличное поле содержит следующие колонки:


·        Акционер (элемент справочника «Организации», «Физические лица» или предопределенный элемент «Неконтролирующие акционеры») – вносится ссылка на инвестора - владельца доли в капитале;

·        Доля в чистых активах, % (доля владения инвестора в чистых активах);

·        Доля привилегированных акций, % (доля инвестора в привилегированных акциях).


Суммарная доля по каждому объекту инвестирования равна 100% (для привилегированных акций может быть либо 100%, либо 0% - если привилегированные акции не выпускались). Корректировка итогового значение долей происходит автоматически за счет предопределенного значения «Неконтролирующие акционеры».


Замечание:

Предопределенный элемент «Неконтролирующие акционеры» нужно вводить последним – тогда корректировка будет сразу отражать истинное положение владельцев.  


Для редактирования табличного поля предназначена панель управления с кнопками «Добавить», «Создать новый элемент копированием текущего», «Изменить», «Удалить», «Переместить вверх», «Переместить вниз», «Обновить состав инвесторов».


Данные табличного поля можно по кнопке «Записать» записать в регистр сведений «Состав инвесторов». По кнопке «Записать и закрыть» элемент справочника будет записан в регистр сведений «Состав инвесторов» и закрыта форма редактирования.
