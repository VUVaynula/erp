Справочник «Способы компоновки источников данных»

 

Справочник «Способы компоновки источников данных» предназначен для хранения настроек формирования отчетов «Универсальный отчет (по источникам данных) и  «Произвольный отчет». Справочник используется, когда необходимо обрабатывать несколько источников данных или данные из одного источника с различными параметрами.

 

На закладке «Колонки результата» задается перечень выводимых колонок в отчет. Задается «Имя колонки» и «Вид колонки». 

 

 На закладке «Источники данных» задается перечень источников данных для формирования отчетов и соответствие колонок данным источников данных. По кнопке «Заполнить» проставляется соответствие колонок результата данным источника данных, сопоставление происходит по имени колонки  (сопоставить можно вручную).

 

На закладке «Элементы данных» задаются «Группировка» элементов с произвольным наименование и минимум один элемент данных с видом «Выборка данных».

 

 Для элемента данных заполняются следующие реквизиты:

·        Наименование – произвольный текст;

·        Вид -  указывается одно из значений «Группировка», «Выборка данных», «Формула»;

·        Кодификатор – может указываться вручную, можно заполнить автоматически, по кодификатору можно в дальнейшем настроить вывод данных в отчет;

·        Коэффициент – по умолчанию равен 1, влияет на вычисление суммы группировки;

·        Закладка «Отбор» – на закладке обязательно указывается источник данных и необходимые отборы из источника данных (закладка доступна при указании вида «Выборка данных»);

·        Закладка «Субконто» - на закладке указывается порядок вывода данных по субконто (закладка доступна при указании вида «Выборка данных»). Например, при выборке данных по разным счетам мы хотим, чтобы первым субконто всегда выводились контрагенты, в этом случае мы задаем порядок вывода субконто.

·        Закладка «Прочее» -  задаются параметры сдвига дат (закладка доступна при указании вида «Выборка данных»);

·        Закладка «Формула» – на закладке задается формула расчета по элементам данных.
