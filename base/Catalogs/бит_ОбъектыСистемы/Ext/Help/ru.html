Справочник «Объекты системы»

Справочник «Объекты системы» предназначен для хранения объектов, на которые распространяется механизм настройки прав доступа.

При добавлении нового объекта конфигурации, он автоматически добавиться в справочник при первом запуске, система проинформирует пользователя о добавлении.
