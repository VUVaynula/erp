Справочник «Виды затрат (МСФО)»


Справочник «Виды затрат (МСФО)» используется для задания структуры затрата для целей дальнейшего использования в расшифровках отчетности.


В том случае, если в справочнике необходимо хранить информацию по структуре затрат нескольких отчетов, то целесообразно выделить в справочнике группы, которые будут соответствовать отчетам, а структуру затрат задать в виде элементов групп.


  • Реквизит «Группа» - указывается группа справочника «Виды затрат (МСФО)»;
  • Реквизит «Наименование» - наименование элемента;
  • Реквизит «Описание» - описание статьи, которое при необходимости выводится в отчетности.
