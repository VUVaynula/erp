Справочник «Рыночные ставки по кредитам и займам»


Справочник (меню «МСФО:Фининструменты» - «Рыночные ставки по кредитам и займам») предназначен для хранения рыночных ставок по кредитам и займам, которые должны быть внесены пользователями Системы перед тем, как начинать отражение финансовых инструментов по МСФО с использованием описываемой ниже функциональности. (Источником информации о рыночных ставках по кредитам может служить, например, «Бюллетень банковской статистики», публикуемый на интернет-сайте ЦБ РФ).


Каждый элемент справочника включает в себя следующие сведения (реквизиты):


  • «Валюта» – валюта кредитов или займов;
  • «Период» – дата, начиная с которой применяются данные значения ставок;
  • В табличной части каждая запись включает в себя колонки:
  • «Число дней до» - максимальная продолжительность срока кредита в днях, при которой применимо данное значение процентной ставки (ставка может быть различной по краткосрочным и по долгосрочным кредитам);
  • «Ставка» - значение рыночной процентной ставки (в процентах годовых).
