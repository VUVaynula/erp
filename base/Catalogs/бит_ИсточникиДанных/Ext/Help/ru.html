Справочник «Источники данных»

 

Справочник «Источники данных» предназначен для хранения настроек заполнения документа «Получение фактических данных»  и настроек формировании отчета «Универсальный отчет (по источникам данных).

 

При  создании пользователь указывает:

·        Группа – родитель текущего элемента;

·        Тип – может принимать значение «Получение факта» - в этом случае используется в документе «Получение фактических данных» или  «Отчетность» - тогда используется в настройках формирования отчета «Универсальный отчет (по источникам данных).

·        Запрос – элемент справочника «Запросы». По кнопке «Заполнить» заполняются по данным запроса закладки «Данные», «Отбор», «Порядок», «Параметры»

·        Данные – для типа источника данных «Получение факта»  проверяются обязательные поля:

·        ЦФО;

·        СтатьяОборотов;

·        Сумма -  результат запроса должен иметь имя «Сумма» или должен быть назначен синоним «Сумма»

·        Отбор – заполняются по данным запроса;

·        Порядок - заполняются по данным запроса;

·        Параметры – заполняются по данным запроса;

·        Прочее – на закладке указывается признак использования «Отключено».

 

Правильность настройки данных можно проверить с помощью вспомогательного отчета «Проверка способа компоновки отчета». Отчет вызывается по кнопке «Проверить».
