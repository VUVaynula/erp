Справочник «Счета бюджета»

 

Справочник «Счета бюджета» предназначен для хранения списка счетов для которых необходимо получать  и выводить начальные остатки при формировании отчетов «Отчет по бюджету» и «План-фактный анализ по бюджету». Справочник является подчиненным справочнику «Бюджеты».  Данные вводятся из формы справочника «Бюджеты».
