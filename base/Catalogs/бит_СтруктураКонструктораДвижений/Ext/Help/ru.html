Cправочник «Структура конструктора движений»


Для того, чтобы привести вид конструктора шаблонов движений в удобный для пользователя вид, предназначен справочник «Структура конструктора движений».


Каждый элемент справочника описывает положение и свойства колонок конструктора для одного из регистров конфигурации. При выборе регистра, для которого производится настройка вида конструктора, система анализирует содержимое справочника «Виды реквизитов проводок» для данного регистра и предлагает пользователю задать расположение, и параметры колонок конструктора. Также при нажатии на кнопку «По умолчанию…» выполняется заполнение параметров по умолчанию.  


Назначение реквизитов и элементов формы справочника:


  • Регистр - регистр, для которого производится настройка структуры конструктора движений. Элемент выбирается из справочника «Объекты доступа»;
  • По умолчанию… - кнопка командной панели. При нажатии на данную кнопку система предлагает настройку расположения колонок и их параметров по умолчанию;
  • Предпросмотр - позволяет пользователю визуально контролировать конечный вид конструктора движений. При изменении положения колонок и их ширины непосредственно в данном поле, система запомнит это расположение;
  • Реквизит - элемент управления панели свойств реквизита. Позволяет выбрать реквизит для настройки его свойств;
  • Синоним -  элемент управления панели свойств реквизита. Позволяет задать представление реквизита движения (наименования колонки конструктора движений);  Положение -  элемент управления панели свойств реквизита. Позволяет задать положение выбранной колонки относительно других колонок;
  • № колонки -  элемент управления панели свойств реквизита. Определяет порядковый номер колонки в табличной части «Предпросмотр»;
  • Ширина -  элемент управления панели свойств реквизита. Позволяет задать ширину колонки, соответствующей выбранному реквизиту;
  • Видимость -  элемент управления панели свойств реквизита. Включает/отключает видимость колонки (реквизита) в конструкторе шаблонов движений.


При создании шаблона движений, системой анализируется справочник «Структура конструктора движений» на предмет наличия настройки внешнего вида конструктора для выбранного регистра. Если таковая отсутствует – колонки таблицы шаблона будут расположены по умолчанию. Если присутствует – колонки таблицы приводятся к виду, определенному в найденном элементе справочника.
